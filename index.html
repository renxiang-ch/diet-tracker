☁ 云同步设置

填入你的 Supabase 项目信息后，所有数据将实时同步到云端，两台设备共用同一份记录。

首次使用？ 在 Supabase 控制台的 SQL Editor 中运行以下语句创建数据表： 
CREATE TABLE kv_store (
  key TEXT PRIMARY KEY,
  value JSONB,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE kv_store ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public access" ON kv_store
  FOR ALL USING (true) WITH CHECK (true);
然后在 Database → Publications → supabase_realtime 中为 kv_store 开启 Realtime。
